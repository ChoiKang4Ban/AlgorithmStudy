# 🙏 PR 규칙

**PR 이름 :**

      필수-[문제번호] 문제이름 (자기이름) (1문제만 풀었을때)

      [문제번호] 문제이름,[문제번호] 문제이름 (자기이름) (2문제 다 풀었을때 (PR 수정 부탁드릴게요!))

      또는

      선택-[문제번호] 문제이름

**PR 내용 :**

      구현어떻게 하였는지
